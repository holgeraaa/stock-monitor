# A股盯盘推送

每小时自动抓取A股行情数据，通过QQ邮箱推送到微信通知。

## 工作原理

```
GitHub Actions 定时触发 (交易日 9:07/10:07/11:07/13:07/14:07)
    ↓
Python 脚本抓取东方财富行情数据
    ↓
通过 QQ 邮箱 SMTP 发送邮件
    ↓
微信收到 QQ 邮箱通知 📱
```

## 标的池

- 通富微电 (002156) - 深市主板
- 长电科技 (600584) - 沪市主板
- 中国石油 (601857) - 沪市主板

## 邮件格式

标题：【盯盘】10:07 沪指3985 +0.38%
正文：沪指点位、板块涨跌、北向资金、标的股价、操作建议

## 注意事项

- 仅推送沪深主板标的，不含创业板/科创板
- 数据来源东方财富公开接口，有3-5分钟延迟
- 仅交易日（周一至周五）推送
